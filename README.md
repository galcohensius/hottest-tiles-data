# Game Analytics - Levels Funnel Analysis

Exploratory data analysis and retention analysis for a mobile puzzle game.

## Contents

- **EDA.py**: Main exploratory data analysis script generating visualizations
- **retention_EDA.py**: Retention and churn analysis script
- **hottestTiles_LevelsFunnel.csv**: Game level funnel data

## Visualizations

The analysis generates several visualizations in the `EDA_plots/` and `retention_plots/` directories:

- Player progression
- Attempts per level
- Churn rates
- Game time analysis
- Retention metrics
- And more...

## Requirements

- Python 3.x
- Install dependencies: `pip install -r requirements.txt`

## Usage

```bash
cd data
python analytics/EDA.py
python analytics/retention_EDA.py
```
